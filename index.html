overview

LogLens is a full-stack backend project built in Python across 5 phases. It ingests raw server log files, parses them using regex, stores structured data in a SQLite database via SQLAlchemy ORM, and exposes everything through a FastAPI REST API with auto-generated Swagger documentation.

The project bridges backend engineering and machine learning — two independent anomaly detectors automatically flag unusual log entries, and everything is accessible through a dark-themed web dashboard and a Click-powered CLI.

features
  • Regex-based log parser that extracts timestamp, level, endpoint, status code, and response time from raw .log files
  • SQLite database storage via SQLAlchemy ORM — no raw SQL written
  • REST API with 6 endpoints built with FastAPI, including file upload and auto-generated Swagger documentation at /docs
  • Dual anomaly detection: Z-score statistical method and Isolation Forest unsupervised ML model (scikit-learn)
  • Dark-themed web dashboard at /dashboard with live stats, charts, log table, and anomaly highlighting
  • CLI tool with 4 commands: loglens parse, loglens logs, loglens summary, loglens anomalies
  • 31 automated tests across all 5 components using pytest fixtures, monkeypatching, FastAPI TestClient, and Click CliRunner
tech stack
Python 3.14 FastAPI SQLAlchemy SQLite scikit-learn Click pytest uvicorn Pydantic Render (hosting)
what makes it stand out
  • Bridges backend engineering and machine learning in a single cohesive project
  • Two independent ML methods cross-validate each other — if both flag the same entry, confidence is higher
  • Isolation Forest is unsupervised — no labeled training data needed, the model learns what "normal" looks like on its own
  • Full test suite with 31 tests — not just functional code but verified code
  • Built phase by phase from scratch, not from a tutorial

how it all started

I was born and raised in Nepal. Being from a third-world country, resources were limited. However, I was lucky enough to be born into an upper-middle-class family, so I was fortunate enough to attend a decent school.

In 7th grade I was able to work with a computer for the first time in the computer lab. I thought they were boring but unique. But it all changed when my friend was able to upload a copy of pirated GTA San Andreas — then, all of a sudden, the same boring piece of tech became the most amazing thing I had ever seen in my entire life.

The real change happened when we relocated to the USA in 2022, where I found out that my relatives were actually well-off software engineers. They inspired me to get into software engineering. Then I learned how easily your day-to-day tasks can become much more efficient when you optimize your daily life with technology and software.
